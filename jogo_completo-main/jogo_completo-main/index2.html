Regras da Fase 3:

  • O objetivo é encontrar pares de palavras que têm sons iguais ou escritas iguais, mas significados diferentes.
  • As cartas começam viradas para baixo.
  • Cada jogador vira duas cartas por vez.
  • Se as palavras formarem um par, o jogador fica com as cartas e joga novamente.
  • Se não formarem um par, as cartas são viradas de volta.
  • O jogador tem 30 segundos para completar o jogo.
  • O jogador deve encontrar todos os pares para vencer no final.

Dicas para o Jogo de Cartas

Aqui estão algumas dicas para ajudá-lo a se preparar:

  • Preste atenção nos sons das palavras. Muitos pares têm som semelhante, mas significados diferentes.
  • Mantenha as cartas na memória e tente lembrar onde os pares estão localizados.
  • Os pares podem ter a mesma escrita ou sons parecidos, então observe bem as palavras.
  • O tempo é limitado! Fique atento aos 30 segundos para encontrar todos os pares.
